## API

参数 | 说明 | 类型 | 默认值
----|------|-----|------
type | 类型 | `loading,line,dot` | `loading`
loadingText | 当 `type==='loading'` 时显示的文本 | `string` | `正在加载`
lineText | 当 `type==='line'` 时显示的文本 | `string` | `暂无数据`
